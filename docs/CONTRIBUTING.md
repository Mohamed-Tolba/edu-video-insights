### How to Contribute to Edu-Video-Insights

1. Fork the repository and clone it locally.
2. Add your video details to `data/video_submission.csv`.
3. Run the extraction pipeline (if applicable).
4. Review the resulting `new_*.csv` files.
5. Submit a pull request with your updates.

Please validate all inputs using the provided scripts in the `scripts/` directory.
